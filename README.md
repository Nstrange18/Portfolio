# Portfolio

Personal overview of my tech life. A modern, responsive portfolio website built to showcase my projects, experience, and skills.

## 🚀 Built With

- **[React](https://react.dev/)**: Frontend library for building the user interface.
- **[Vite](https://vitejs.dev/)**: Next-generation frontend tooling for fast development and building.
- **[Tailwind CSS](https://tailwindcss.com/)**: Utility-first CSS framework for rapid UI styling.
- **[Framer Motion](https://www.framer.com/motion/)**: For smooth, professional animations.
- **[React Router](https://reactrouter.com/)**: For seamless client-side routing.
- **[@iconify/react](https://iconify.design/)**: For versatile SVG icons.

## 📦 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Nstrange18/Portfolio.git
```

2. Navigate into the project directory:
```bash
cd portfolio
```

3. Install the dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`.

## 📁 Project Structure

- `src/components/`: Reusable UI components (Navbar, Footer, ProjectCard, etc.)
- `src/Pages/`: Primary page views (Hero, About, Projects, Experience, Contact)
- `src/Layout/`: Application layout wrappers
